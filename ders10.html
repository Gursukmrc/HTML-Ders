Etiketlerine Ekstra Ozellikler Eklemek

Bu ek ozellikler Turkce kaynaklarda HTML etiketlerinin nitelikleri,ozellikleri veya oznitelikleri diye gecmektedir.

Ozellikleri:

  1. Tum HTML Elementleri nitelik alabilirler.
  2. Her zzaman baslangic elemaninin icine yazilir.
  1. Bu nitelikler,etiketler konusunda ekstra ozellik bilgisi saglar.
  2. Kelimeler Arasinda tire isareti koyarak (html-etiketi) veya camel case(htmlEtiketi) seklinde kullanilabilir. HTML'de genellikle -(tire) kullanilir.
  3. class ve id gibi ozniteliklerin ismi belirtilirken turkce karakter tercih edilmemelidir.
  4. Kullanabilmek icin ozniteligi yazdiktan sonra ="" kullanip istedigimiz ozelligi tirnaklarin icine yazmaliyiz.

Her etikete id ve class ozniteliklerini ekleyebiliriz.

id(identity=kimlik

id,kullanilan etiketin kimligini belirtir. Biriciktir (Unique) yani dokumanin icindeki bir etikete yalnizca bir kez o id ismi verilebilir.
Buradaki id yi kullanarak direkt olarak bu HTML etiketine ulasabilirsin

class

Class,HTML etiketinin hangi sinifta oldugunu belirmemizi saglar. Ayni sinif adini Birden Fazla HTML etiketine verebiliriz.
Burada goruldugu gibi iki HTML etiketi de (img ve a) documents sinifinda bulunmaktadir. Daha sonrasinda ayni sinifta bulunan iki etikete CSS ile ayni anda degisiklik yapilabilir.
